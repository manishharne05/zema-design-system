# Alert

### Alerts are components used to provide contextual feedback and notifications in response to user actions or system activities.

---
<br />

## When to use:
- It is necessary to show contextual information.
- For urgent feedback.

<br />

## When not to use:
- Do not cover other interface elements with an Alert.
- It is necessary to display information in a non-intrusive way. In this case, we recommend using b-toasts.
- It is necessary to show extensive contextual information. In this case, we recommend using AlertCard.
