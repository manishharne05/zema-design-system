# Spinner

### Spinners is a component used to indicate that the progress of an operation has not yet been completed.
---
<br />

## When to use:
- It is necessary to prevent the viewing and navigation of page content while there is a pending operation.

<br />

## When not to use:
- Viewing and navigation cannot be interrupted.
