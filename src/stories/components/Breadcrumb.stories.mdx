# Breadcrumb

### Breadcrumbs are used to indicate to the user the path taken between the pages of an application until reaching the current page.
---
<br />

## When to use:
- When there are two or more levels in the page hierarchy;
- When you need to tell the user which page they are on and how they can retrace the steps that took them to the current page;

<br />

## When not to use:
- In cases where there is only one page in the navigation structure.

<br />

## Note:
- Ideally Breadcrumbs should be used with <b>three</b> or more hierarchy levels. However, the Breadcrumb of
RDS can be used with <b>two</b> levels because in this scenario it behaves like a return button to the previous page.
