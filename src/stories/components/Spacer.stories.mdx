# Spacer

### Auxiliary component used to add paddings and margins following the RDS token standard.
---
<br />

## When to use:
- In case a certain component or interface element only needs spacing (padding or margin) how to modify the style.

<br />

## When not to use:
- In cases where a given component requires several style modifiers. For these cases
use classes and include paddings and margins with RDS spacing tokens.

<br />

## Comments:
- ⚠️ Whenever possible, give preference to the use of tokens over Spacer.
- The value sent to Spacer props is always multiplied by 4, just like in RDS scss tokens.
- The maximum accepted value is 12.
- Spacer does not implement negative margins and paddings.
