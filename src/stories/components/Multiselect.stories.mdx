# Multiselect

### Multiselects are components that allow the user to select multiple options divided into categories.
---
<br />

## When to use:
- There is the possibility of selecting multiple items in a select.
- If you want to classify select items into categories.

<br />

## When not to use:
- Select items cannot be classified into categories.
- There is a need to detail the selected options in the input, as we show as a summary only how many and not which options were chosen.

<br />

## Comments:
- RDS Multiselect is a wrapper around <a href="https://vue-multiselect.js.org/">Vue Multiselect</a>
with its compliance with our Principles and recommendations. For detailed information about the component, we recommend
reading the documentation.
