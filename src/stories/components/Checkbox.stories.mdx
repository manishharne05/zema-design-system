# Checkbox

### Checkboxes are form components that allow binary and multiple selections.
---
<br />

## When to use:
- When you need a form component for binary selections.

<br />

## When not to use:
- Do not use checkboxes without labels.
- Avoid negative labels, they are more difficult to interpret.
- When the effect of interacting with Checkbox is required to be immediate. Checkboxes should 
always be used in conjunction with a submit button. To work around cases like this, use Switches.
- Avoid using horizontally aligned checkboxes.
