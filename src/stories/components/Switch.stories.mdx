# Switch

### Switches are components used to activate or deactivate specific settings or options. 

---
<br />

## When to use:
- Configuration controlled by the Switch must apply changes automatically.
- When using labels. When used together, they must describe two states, dependent on the state of the Switch.

<br />

## When not to use:
- In forms that have a submit button.
- Use Switches to control actions that depend on requests without indicating the waiting time to the user.
