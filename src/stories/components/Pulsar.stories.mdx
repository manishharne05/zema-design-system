# Pulsar

### Pulsars are educational components used to instruct users about new features in the interface or product.
---
<br />

## When to use:
- To attract the user's attention to a new feature or area of ​​the interface in a non-intrusive way.


<br />

## When not to use:
- As the only resource for communicating about changes to the interface or product.
- When another pulsar is already in use on the interface.

<br />

## Comments:
- Pulsars should not be a permanent part of the interface. After a certain feature of the 
product has been absorbed/understood by users, the pulse loses its purpose and should be considered removing it.
