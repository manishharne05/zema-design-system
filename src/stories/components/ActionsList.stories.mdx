# ActionsList

### Actions lists are components used to show a list of actions online, some of which may be hidden.
---
<br />

## When to use:
- There are actions associated with an item in a list or table row.
- There are several possible actions in a given situation, but some of them can be collapsed.

<br />

## When not to use:
- The use of buttons is necessary.
- It is necessary to build navigation components (Menu, navbar, etc.).
- The actions listed affect the entire view or more than one set of related data.
