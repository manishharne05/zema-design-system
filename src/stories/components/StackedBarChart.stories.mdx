# StackedBarChart

### A reusable chart component to display data visually.
---
<br />

## When to use:
- Display data in visual formats;
- Comparison of data as a whole;
- Trends over time.

<br />

## When not to use:
- For simple information that can be displayed in a tabular or text format;
- Absolute comparison;
- Many categories.

## Preview

### __Stacked monochrome bar__

When the `variant` prop is specified and there is no color property indicated in the configuration object,
the graph is generated in monochrome mode, with the variant shades changing for each dataset.

### __Stacked bar with multiple colors__

To generate graphs with multiple colors, simply send the `variant` property in each `dataset`.
The shades get darker with each dataset until reaching 700, when they then decrease again until shade 50, when the cycle is repeated.

⚠️ When specifying variants via the configuration object, the variant by prop will be overwritten
