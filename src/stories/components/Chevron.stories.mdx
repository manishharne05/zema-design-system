# Chevron

### Chevrons are graphic elements used in interfaces to indicate the possibility of interaction.
---
<br />

## When to use:
- Part of an element can be hidden from view,
- Indicate actions such as `play`, `next` and `fastfoward`


<br />

## When not to use:
- Like icons.
- Avoid using chevrons without the  `animate` active property
