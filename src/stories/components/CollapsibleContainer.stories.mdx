# CollapsibleContainer

### CollapsibleContainers are components used to reduce vertical space when there is a large amount of information, with the possibility of being expanded to display summarized content.
---
<br />

## When to use:
- It is necessary to optimize vertical space.
- If you want to show summarized items.
- If you have a large amount of information that can be grouped under a topic/title.

<br />

## When not to use:
- There are few items to be summarized;
