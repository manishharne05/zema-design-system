# PageHeader

### PageHeader are components used to add title, subtitle and actions to pages.
---
<br />

## When to use:
- Need to place headers on pages.
- Need to add action buttons at the page level.
- Need to add informative cards at the top of pages.

<br />

## When not to use:
- Off the top of pages.

<br />

## Comments:
- ⚠️ When a URL is placed as part of the subtitle, it is rendered as a link in the component. To do so, the URL must be preceded by http:// or https://.
