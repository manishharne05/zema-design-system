# CarouselController

### CarouselControllers are components used to control the display of elements in carousels.
---
<br />

## When to use:
- You have a category of interface elements that are repeated and can be displayed in the form of a carousel.

<br />

## When not to use:
- You are working with tables. In these cases we recommend using pagination (TODO).
- All content to be managed can be displayed at once. In this case there is no need to control pages and use the component;
