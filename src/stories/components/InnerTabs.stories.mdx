# InnerTabs

### InnerTabs are components used to create integrated navigation solutions <a href="https://rds-design-system.netlify.app/?path=/docs/componentes-navega%C3%A7%C3%A3o-tabs--tabs ">Tabs</a>.
---
<br />

## When to use:
- Within tabs, there is a need for another level in the navigation structure;
- In cases where it is very important to show the entire title of the tabs;
- It is necessary to separate the content into tabs and navigate through them;
- It is not the case to use solutions such as navbars, sidebars, cards or links as navigation;

<br />

## When not to use:
- There are too many items to be shown as tabs.
- When the click action performed by the user takes him to another page.

<br />
