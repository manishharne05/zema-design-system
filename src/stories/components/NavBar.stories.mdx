# NavBar

### NavBars are components used to create navigation solutions and menus.
---
<br />

## We recommend using NavBars when:
- It is necessary to separate the content by pages and navigate through them.
- It is not the case to use solutions such as tabs, sidebars, cards or links as navigation.

<br />

## We do not recommend using NavBars when:
- There are too many items to be shown in the menu.
- It is necessary to show the links vertically.
