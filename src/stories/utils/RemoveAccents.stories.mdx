# RemoveAccents()

Sanitizes strings by removing accents.

<hr />
<br />

⚠️ *Method only available for internal development in RDS*

<br />

#### Arguments

**(String):**
- String whose value you want to sanitize
- Mandatory: *Yes*
- Valid values: any string;

<br />

#### Return

**(Strgin):** String without accents.
